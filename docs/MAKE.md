## Linux
Установить его не составить проблем, выполнив следующую команду,
например, если у вас пакетный менеджер apt так:

```bash
sudo apt install make
```

Есть еще несколько утилит, но эта покрывает все мои потребности.

Если у вас ОС, не базируется на Debian, используйте свой пакетный менеджер.
