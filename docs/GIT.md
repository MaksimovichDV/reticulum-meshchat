# Установка GIT

## Windows

На windows установить git не составить труда, просто [скачайте](https://git-scm.com/install/windows) инсталлятор 
с официального сайта и запустите его.

## Linux

На Linux, он, как правило, установлен, можно поверить запустив команду в консоли:

```bash
git --vertion
```

Вывод будет похож на этот:

```bash
git version 2.43.0
```

Если, если он все-таки не установлен, то установить его не составить проблем, выполнив следующую команду, 
например, если у вас пакетный менеджер apt так:

```bash
sudo apt install git
```

Если у вас ОС, не базируется на Debian, используйте свой пакетный менеджер.
